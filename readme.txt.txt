# Quote Generator App

## Description

This is a Quote Generator App built using React. The application fetches random quotes from the [AdviceSlip API](https://adviceslip.com/) and displays them to the user. It uses Axios for making HTTP requests to the API.

## Features

- Fetches random quotes from the AdviceSlip API. 
## Technologies Used

- **React**: Frontend library for building the user interface.
- **Axios**: Promise-based HTTP client for making API requests.
- **AdviceSlip API**: Provides random quotes and advice.

## Installation

To get started, follow these steps:

- Install Node modules:
  Run npm install (or npm i for short) in your terminal to install the required dependencies.

- Start the development server:
  Run npm start to start the development server.

  You should now be up and running!
